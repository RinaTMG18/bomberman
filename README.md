








Jag har gjort ett spel i Unity som heter Bomberman. I spelet styr man en figur som kan gå runt på en bana. Man kan lägga bomber som sprängs efter några sekunder. Explosionen kan förstöra väggar och ta bort fiender. Målet är att besegra fienderna och klara banan utan att bli träffad av explosioner.
Jag har fixat spelet i Unity genom att förbättra funktioner och se till att allt fungerar som det ska, till exempel rörelse, bomber och kollisioner.
I spelet kan man få power-ups som gör spelaren starkare, till exempel
Extra bomber (man kan lägga fler bomber samtidigt)
Speed (man rör sig snabbare)
